Frequently asked questions

  • Copilots are chat-based generative AI interactions from Microsoft. They have specialized functionality based on their users and use cases. Some Copilots are experienced standalone, while others are integrated into Microsoft products, services, and devices.

    Included in this portfolio is Microsoft Copilot, Microsoft 365 Copilot, Microsoft Security Copilot, and GitHub Copilot, among other generative AI solutions and experiences.
  • Microsoft Copilot is your digital companion for daily life designed to inform, entertain, and inspire. Using advanced AI, Copilot understands your questions and requests, provides direct answers, assists with writing, and even creates images.

    Microsoft 365 Copilot is an AI assistant for work that’s built to support organizations and businesses. Copilot combines advanced language models with your business data and your everyday productivity apps—such as Word, Excel, PowerPoint, Outlook, and Teams.

    Microsoft Security Copilot is an AI assistant for daily operations in security and IT—enabling teams to protect their organizations at the speed and scale of AI while maintaining compliance with responsible AI principles.

    GitHub Copilot is an AI coding assistant that helps you write code faster and with less effort, allowing you to focus more energy on problem solving and collaboration.
  • Microsoft offers a range of Copilots tailored for both work and life. Pricing and the licensing requirements depend on the specific solution and how you use it.

    For work, you can try Microsoft 365 Copilot Chat—which is included with eligible Microsoft 365 subscriptions and offers enterprise-grade security and privacy.

    For personal use, you can try Microsoft Copilot on the web for free or download the app.​

    Learn more about Microsoft 365 Copilot licensing, Microsoft Security Copilot licensing, and GitHub Copilot licensing.
  • Microsoft Copilot remembers details that matter to your daily life while keeping your personal data secure and protecting your privacy. You can always manage your privacy preferences in your Copilot Privacy settings.

    When you’re using Copilots at work, all your existing security and compliance requirements are inherited, so only people with the right permissions can access the content it generates. Learn more about data, privacy, and security for Microsoft 365 Copilot.

    As a company, Microsoft is committed to providing trustworthy AI—with decades of research, customer feedback, and learnings informing the capabilities that deliver privacy, safety, and security. Learn more about Microsoft’s trustworthy AI commitments.
  • If you’re looking to use Copilot outside of work, you can start using Microsoft Copilot for free right away at copilot.microsoft.com or by downloading the Microsoft Copilot app on your mobile device. 

    If you want to use Microsoft 365 Copilot at work, you’ll need to have an existing license for a qualifying Microsoft 365 plan and a license for Microsoft 365 Copilot. If you’re not an existing customer, you need to buy an eligible Microsoft 365 subscription. If you’re just starting out, you can also use Microsoft 365 Copilot Chat—an AI chat available at no additional cost with eligible Microsoft 365 subscriptions.

    Additional business and enterprise licenses for GitHub Copilot are available, as well as licensing plans for Microsoft Security Copilot.
  • Use the Copilot learning hub to find trainings, documentation, and videos, and check out Microsoft Adoption for resources on implementing Microsoft 365 Copilot at your organization. 

    You can also find business best practices, resources, and insights here to get the most out of using Microsoft 365 Copilot.
  • To have full access to Copilot in Word, Excel, PowerPoint, and Teams, you’ll need to have a license for a qualifying Microsoft 365 plan and a license for Microsoft 365 Copilot—even if you already use Copilot outside of work or have limited access in Word, Excel, PowerPoint, and Teams.

    Here are some ways you can use Copilot in Word, Excel, PowerPoint, and Teams:
     
    • Writing and editing. Copilot in Word can provide prompts to help you start writing quickly, help edit existing documents, and help trim down and summarize longer texts.
    • Data analysis. Copilot in Excel can help analyze data, write formulas, and organize unstructured data into a more digestible format.
    • Creating presentations. Copilot in PowerPoint can help build presentations, edit notes, and rearrange slides.
    • Streamlining collaboration. Copilot in Teams can summarize meetings, provide transcripts, and share insights from discussions.
